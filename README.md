# Simple blog system by Flask

## What's this?

Python + Flask によるシンプルなブログシステム

***

## Environment

- Python: `3.6.8`
    - Flask(WebFramework): `1.0.3`
        ```bash
        $ pip install flask
        ```
    - uWSGI(ServerProtocol): `2.0.18`
        ```bash
        $ pip install uwsgi

        # in Anaconda environment
        $ conda install -c conda-forge uwsgi
        ```
- Nginx(WebServer): `1.16.0`
